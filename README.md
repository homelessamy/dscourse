# Customs Fraud Detection Pipeline

This repository contains a  implementation of a binary classification pipeline for customs fraud detection. The code:

- Ingests training and test CSV files and validates schema assumptions  
- Applies reproducible, vectorized feature engineering and preprocessing within sklearn objects  
- Trains and tunes multiple classification models using cross-validation and a unified evaluation engine  
- Generates comparison reports, diagnostic visualizations, and a final submission file for deployment and competition use  

> **Note:** Due to course policies and dataset confidentiality, the original data, trained model artifacts, and all quantitative results (e.g., metrics, plots, and intermediate outputs) are not included and cannot be shared in this repository.
